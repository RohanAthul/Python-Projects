
# Band name generator

A simple Python command-line application that generates a unique band name by combining the user's hometown and the name of a pet.

## Description

The Band Name Generator is a beginner-friendly Python script designed to practice basic programming concepts like string manipulation, user input, and f-strings. It provides a fun, interactive way to create a catchy name for your next musical project.

## Running the App

This script is compatible with any operating system (Windows, macOS, Linux) that has Python 3 installed.

- Open your Terminal or Command Prompt.

- Navigate to the folder where you saved main.py.

- Run the script using one of the following commands:

    - python BandNameGenerator.py
